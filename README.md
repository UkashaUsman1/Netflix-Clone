# Netflix Clone (HTML & CSS)

A fully responsive **Netflix Clone** built using **HTML** and **CSS**.

## 🌟 Features

- ✅ **Responsive Design**: Optimized for all screen sizes (mobile, tablet, desktop).
- ✅ **User Interface**: Mimics the layout of Netflix, including movie and TV show categories, a header with navigation, and a footer.
- ✅ **CSS Grid and Flexbox**: Used to create a flexible and modern layout that adjusts based on the viewport size.
- ✅ **Styling**: Clean and modern CSS to replicate the Netflix aesthetic.
